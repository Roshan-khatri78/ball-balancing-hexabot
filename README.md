🎯 What is ASTRA?
A smart hexagonal platform that uses AI-powered control to balance a ball perfectly. The system learns and adapts in real-time using computer vision and machine learning.
Key Features:

🧠 AI-powered adaptive PID control
📹 Real-time computer vision tracking
🎮 Interactive TFT display interface
⚡ Dual ESP32 architecture
📊 Live performance visualization

🎯 Why Upgrade to Steppers?
Phase 1 (Servos) proved the concept ✅
Phase 2 (Steppers) maximizes precision 🎯
4 Key Improvements:

20x better accuracy (±0.05° vs ±1-2°)
Zero position drift (rock-solid holding)
Smoother control (6400 steps/revolution)
Better for AI (predictable response)

Trade-off: 2.5x higher cost but worth it for research quality
<br>
💡 Which Should I Build?
Build Phase 1 if:

First time with ball balancing
Budget under $50
Want to learn the basics

Build Phase 2 if:

Need research-grade results
Want maximum precision
Budget allows ~$100


Both work great! Choose based on your goals.
